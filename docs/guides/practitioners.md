## For Working Professionals

This guide helps practitioners apply the HUMMBL Framework in professional contexts.

---

## Getting Started

**Immediate Actions:**
1. Assess 3 current work problems using [wickedness scoring](../core/wickedness-scoring.md)
2. Review [problem complexity tiers](../core/problem-complexity-tiers.md)
3. Start with [Base6-Base12 models](../core/base-n-architecture.md)
4. Apply tier-matched approaches

---

## By Professional Context

### Project Managers

**Typical Problems:** Tier 2-3 (Complicated to Complex)

**Key Skills:**
- Assess project complexity systematically
- Match methodology to tier
- Manage stakeholder expectations
- Adapt when complexity shifts

**Recommended Base-N:** Base12-Base24

---

### Team Leaders

**Typical Problems:** Tier 2-3 (Complicated to Complex)

**Key Skills:**
- Assess team challenges
- Navigate interpersonal complexity
- Build adaptive capacity
- Foster learning culture

**Recommended Base-N:** Base12-Base24

---

### Consultants & Advisors

**Typical Problems:** Tier 3-4 (Complex to Wicked)

**Key Skills:**
- Rapid complexity assessment
- Multi-stakeholder navigation
- Framework customization
- Teaching others the framework

**Recommended Base-N:** Base24-Base36

---

### Senior Executives

**Typical Problems:** Tier 3-5 (Complex to Super-Wicked)

**Key Skills:**
- Strategic complexity assessment
- Organizational transformation
- Multi-system interventions
- Long-term vision with uncertainty

**Recommended Base-N:** Base24-BASE120

---

## Professional Development Path

### Phase 1: Foundation (3-6 months)
- Master wickedness scoring
- Learn Base6-Base12 models
- Apply to simple/complicated problems
- Build systematic assessment habit

### Phase 2: Integration (6-12 months)
- Expand to Base24 models
- Tackle complex problems
- Develop personal style
- Teach colleagues basics

### Phase 3: Mastery (1-3 years)
- Master Base24-Base36 models
- Handle wicked problems
- Create organizational approaches
- Lead transformations

### Phase 4: Expertise (3+ years)
- Contribute to framework development
- Publish case studies
- Mentor advanced practitioners
- Extend to new domains
